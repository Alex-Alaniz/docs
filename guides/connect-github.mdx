Connect GitHub to let your AI assistant manage your repositories, issues, and starred repos.

## Steps

<Steps>
  <Step title="Go to Toolkits">
    Navigate to **Dashboard → Toolkits** and search for "GitHub".
  </Step>
  <Step title="Click Connect">
    Click **Connect** on the GitHub card. You'll be redirected to GitHub's OAuth page.
  </Step>
  <Step title="Authorize">
    Authorize the Composio GitHub app. Choose which repos to grant access to.
  </Step>
  <Step title="Verify">
    The GitHub card should show **Active**.
  </Step>
  <Step title="Test">
    Ask your AI: *"List my GitHub repositories"* or *"What repos have I starred recently?"*
  </Step>
</Steps>

## Available Actions

| Action | Description |
|--------|-------------|
| List starred repos | Browse your starred repositories |
| Get repo details | View repo metadata, README, and stats |
| List issues | View open issues on a repository |
| Create issue | Open a new issue |
| Search code | Search across your repositories |
