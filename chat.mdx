The Chat tab provides a direct conversation interface with your AI assistant.

## Features

- **Message thread** with streaming responses and elapsed timer
- **Session selector** — switch between conversation contexts or start new ones
- **Thinking toggle** — show/hide the AI's reasoning process
- **Focus mode** — fullscreen chat with all UI chrome hidden
- **Image paste** — paste images directly from clipboard
- **Message queue** when assistant is busy processing
- **Stop button** to abort long-running responses

## Quick Messages

From the OpenClaws Dashboard, you can send quick messages without opening the full Control UI. The chat input at the bottom of the Dashboard proxies messages to your gateway.

<Tip>
  For the full experience with session management, thinking toggle, and focus mode, use the **Control UI** directly.
</Tip>
